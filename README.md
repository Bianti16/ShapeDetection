# ShapeDetection
## Introduction
A computer vision program that detects shapes in a video and uses machine learning to classify them as circles, squares, or triangles.
## Technologies:
• Python 3.7    
• TensorFlow 2.1.0    
• Scikit-learn 0.23.2
• OpenCV 4.4.0    
• Kivy 2.0.0    
• NumPy 1.19.1    
• PIL 7.2.0
